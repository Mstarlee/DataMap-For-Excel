兼容性和安装装注意事项：
DataMap For Excel，数据地图，兼容MicroSoft Excel 2010-2021，32/64位，Win7-Win11。
与Foxit的EXCEL插件有冲突，安装和使用时注意禁用Foxit的EXCEL插件。

腾讯的电脑管家会对插件使用的eo.webbrowser拦截，请手动添加到信任区。
注意：需卸载旧版再安装新版

2023.05.04 V6.9.2
主要升级及变更内容：
对插件的程序做了大范围更新，优化性能和稳定性，去除bug
【1】文件类的资源（如模板缩略图）改由本地数据库加载，加载更迅速，体验更好；
【2】增加定义区域，基于高德地图绘制区域多边形，可以保存，可以转换成SVG插入到Excel中，可以作为区域边界在eCharts中使用
【3】优化eCharts转换SVG功能，并增加了将生成的SVG直接插入到Excel的功能（Excel2016以上版本有效）；
【4】优化图标库，定义了分类，内置部分图标；
【5】优化设置颜色，增加了3项底色和颜色值互转功能；
【6】优化eCharts下的选择区域功能，更迅速，不再依赖地图key；
【7】优化图表参数，代码窗口输出的代码，自动排版；
【8】删除高度地图自定义背景功能；
【9】删除分享功能（上传数据，生成二维码）；
【10】其他功能优化和bug修复
